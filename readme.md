# DothiVang Portal

## Project

Create a modern enterprise admin dashboard for a Vietnamese jewelry wholesale company named DothiVang.

This is NOT an ecommerce website.

This is an internal business management system.

The users are:

- Administrator
- Sales
- Production Manager
- Warehouse
- Accountant
- Marketing
- Customer Service

Use only HTML5, CSS3 and vanilla JavaScript.

Do not use React.

Do not use Bootstrap.

Use TailwindCSS CDN.

Create reusable UI components.

---

# Design Style

Use Ant Design style.

Very clean.

Modern.

Minimal.

Professional.

Enterprise SaaS.

Inspired by

- Ant Design Pro
- Vercel Dashboard
- Stripe Dashboard
- Linear

Do not use colorful admin templates.

No gradient.

No glassmorphism.

No neumorphism.

Use white background.

Light gray borders.

Gold as primary color.

---

# Layout

The admin layout contains

- Left Sidebar
- Top Header
- Main Content

Sidebar is collapsible.

Header contains

- Search
- Notifications
- User Profile
